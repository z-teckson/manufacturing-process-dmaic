# Manufacturing Process DMAIC Project

## Project Charter

**Objective:** Optimize the widget assembly line to reduce cycle time by 10% within the next quarter.

**Scope:** The project focuses on the widget assembly process from raw material receipt to final quality inspection and shipping.

**Goals:**
- Reduce overall cycle time by 10% (from current average of 120 minutes to 108 minutes)
- Decrease defect rate from 15% to less than 5%
- Improve operator productivity by eliminating non-value-added waiting time

**Team:** Lean Six Sigma Green Belt practitioner, Process Engineer, Production Supervisor

**Timeline:** 3 months (aligned with DMAIC phases)

**Success Metrics:**
- Cycle time measurements
- Defect counts per shift
- Operator satisfaction survey
